SQL Data Analysis project on the Chinook Music Store database using PostgreSQL. This project includes beginner, intermediate, and advanced SQL queries for business insights.

## 📌 Project Overview
This project analyzes the Chinook Music Store database using SQL. It demonstrates SQL skills by solving real-world business problems using PostgreSQL.

## 🛠️ Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

## 🗄️ Database
Chinook Music Store Database

## 📊 SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions
- JOIN
- Subqueries
- Common Table Expressions (CTE)
- Window Functions

## 📋 Business Questions Solved
1. Find the senior-most employee.
2. Find the countries with the most invoices.
3. Display the top 3 invoice totals.
4. Find the city with the highest total invoice amount.
5. Identify the best customer.
6. List all Rock music listeners.
7. Find the top 10 Rock artists.
8. Find tracks longer than the average track length.
9. Calculate the amount spent by each customer on artists.
10. Find the most popular music genre in each country.
11. Find the highest-spending customer in each country.
